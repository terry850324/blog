VB.NET SMTP寄送Mail

發表於 2020-11-16 更新於 2020-11-30 分類於 工作筆記

透過System.Net.Mail.SmtpClient類別寄送Mail。

引用參考

Imports System.Net.Mail  'SMTP主要類別
Imports System.Net       '帳號驗證

使用參數

System.Net.Mail.MailMessage

寄件者
MailMessage.From

型態為MailAddress，使用New MailAddress(“Mail Address”)建立。

收件者
MailMessage.To

為MailAddress集合，設定收件者要使用Add()方法。

主旨與內容
MailMessage.Subject  '信件主旨
MailMessage.Body     '信件內容

System.Net.Mail.SmtpClient

伺服器資訊
SmtpClient.Host  'SMTP伺服器
SmtpClient.Port  'SMTP伺服器使用的埠號

設定提供SMTP服務的伺服器資訊。

使用者資訊
SmtpClient.Credentials

其實就是Mail服務的帳號和密碼，要使用System.Net中的NetworkCredential建立。

完整 Function

Public Sub SendMail(ByVal str_MailAct As String, ByVal str_MailPwd As String,
                    ByVal strlst_MailToAddress As List(Of String),
                    ByVal str_Subject As String, ByVal str_MailBody As String)

    Dim obj_SmtpClient As New SmtpClient
    Dim obj_MailMessage As New MailMessage

    Try

        ' Mail content
        obj_MailMessage.From = New MailAddress(str_MailAct)
        For Each str_MailAddress As String In strlst_MailToAddress
            obj_MailMessage.To.Add(str_MailAddress)
        Next
        obj_MailMessage.Subject = str_Subject
        obj_MailMessage.IsBodyHtml = True
        obj_MailMessage.Body = str_MailBody

        ' SMTP setting
        obj_SmtpClient.Host = "smtp.gmail.com"
        obj_SmtpClient.Port = "587"
        obj_SmtpClient.Credentials = New NetworkCredential(str_MailAct, str_MailPwd)

        ' Send mail
        obj_SmtpClient.Send(obj_MailMessage)

    Catch ex As Exception
        MsgBox("Sending mail failed: " & ex.ToString, vbExclamation)
    End Try

End Sub

註解

MailMessage用在最後寄送Mail時傳入SmtpClient.Send()方法，然而SmtpClient.Send()有另一個多載，可以將MailMessage中設定的「寄件者」、「收件者」、「信件主旨」、「信件內容」依序傳入下方多載：

SmtpClient.Send(String, String, String, String)

所以不一定需要宣告MailMessage物件。
